# TableCRM Mobile Order

Мобильная форма оформления заказа для TableCRM в формате webapp.

## Технологии

- **Next.js 14** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **React Context** - управление состоянием

## Функциональность

- ✅ Авторизация по токену
- ✅ Поиск клиента по телефону
- ✅ Выбор счета (paybox)
- ✅ Выбор организации
- ✅ Выбор склада
- ✅ Выбор типа цены
- ✅ Выбор товаров с добавлением в корзину
- ✅ Создание продажи
- ✅ Создание и проведение продажи

## Требования

- **Node.js** >= 14.18.0 (рекомендуется >= 18.17.0)
- **npm** >= 6.0.0

> **Примечание:** Если у вас Node.js версии 14.x, проект использует Next.js 13, который совместим с этой версией. Для лучшей производительности рекомендуется обновить Node.js до версии 18 или выше.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Сборка для продакшена

```bash
npm run build
npm start
```

## API Endpoints

Приложение использует следующие endpoints TableCRM API:

- `/api/v1/contragents` - поиск клиентов
- `/api/v1/payboxes` - получение счетов
- `/api/v1/organizations` - получение организаций
- `/api/v1/warehouses` - получение складов
- `/api/v1/price_types` - получение типов цен
- `/api/v1/nomenclature` - получение товаров
- `/api/v1/docs_sales` - создание продажи

## Структура проекта

```
├── app/
│   ├── layout.tsx          # Главный layout
│   ├── page.tsx            # Главная страница
│   └── globals.css         # Глобальные стили
├── components/
│   ├── TokenForm.tsx       # Форма авторизации
│   ├── ClientSearch.tsx    # Поиск клиента
│   ├── SelectField.tsx     # Компонент выбора
│   ├── ProductSelector.tsx # Выбор товаров
│   └── OrderForm.tsx       # Главная форма заказа
├── context/
│   └── AppContext.tsx      # Context для состояния
└── services/
    └── api.ts              # API сервисы
```

## Использование

1. Введите токен для авторизации в конкретной кассе
2. Введите телефон клиента и найдите его
3. Выберите счет, организацию, склад и тип цены
4. Добавьте товары в корзину
5. Нажмите "Создать продажу" или "Создать и провести"

# Тестовые номера телефонов для проверки приложения

API успешно работает с токеном: `af1874616430e04cfd4bce30035789907e899fc7c3a1a4bb27254828ff304a77`

## Примеры номеров телефонов для тестирования:

1. **+987123141245** - клиент с ID 797334
2. **0987123141245** - клиент "Holy Koaq" с ID 797333
3. **78900776655** - клиент "Holy Koaq" с ID 797243
4. **+79900776655** - клиент "a" с ID 797242

## Как использовать:

1. Введите токен: `af1874616430e04cfd4bce30035789907e899fc7c3a1a4bb27254828ff304a77`
2. В поле поиска клиента введите любой из номеров выше (можно частично, например: `987`, `799`, `789`)
3. Приложение автоматически найдет совпадения

## Статистика API:

- Всего клиентов: 4041
- API возвращает данные в формате: `{ count: number, result: array }`
